# ArduinoMotorControl

Arduino-based stepper motor control for astronomy projects
(focuser, rotator, alt/az).

## Hardware
- Arduino Uno R3
- DRV8825 stepper driver(s)
- External 12V motor supply

## Goals
- Reliable step timing
- Simple serial protocol
- Integration with Raspberry Pi / NINA

## Status
Early bring-up and motor testing.
